# 🥊 Fight Detection using Deep Learning

This project is a **video classification system** that detects whether a fight is happening in a video or not using Deep Learning.

---

## 🚀 Features

* Detects **Fight vs Non-Fight** scenes
* Uses **CNN-based Deep Learning model**
* Works on **video input**
* Frame extraction and preprocessing included

---

## 🧠 Tech Stack

* Python
* TensorFlow / Keras
* OpenCV
* NumPy

---

## 📂 Project Structure

```
fight-detection-aiml/
│
├── model/              # Trained model
├── src/                # Source code
├── data/               # Dataset (not included in repo)
├── README.md

```

---

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/fight-detection-aiml.git
cd fight-detection-aiml
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

---

## 📊 How It Works

1. Extract frames from videos
2. Label frames as fight / non-fight
3. Train CNN model
4. Test on new videos

---

## ▶️ Usage

### Train Model

```bash
python src/train.py
```

### Test Model

```bash
python src/test.py
```
